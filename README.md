## 🔍 Project Overview

**MEDFIND** is a comprehensive web application designed to simplify the healthcare journey. Inspired by platforms like `bddoctors.com`, it serves as a centralized hub where users can:

- Find hospitals, doctors, and diagnostic labs based on their location.
- View detailed profiles, including specialties, ratings, and availability.
- Book appointments seamlessly.
- Access medical history and reports.

The application features distinct portals for **Patients**, **Doctors**, **Administrators**, **Pharmacies**, and **Laboratories**, ensuring a tailored experience for every user role. This frontend repository demonstrates a modular, scalable, and responsive architecture built with vanilla HTML, CSS, and JavaScript.

---

## ✨ Key Features

- **Role-Based Dashboards:** Dedicated interfaces for Admin, Doctor, Patient, Pharmacy, and Lab staff.
- **Location-Centric Search:** Search for healthcare providers by city and area with intelligent filtering.
- **Comprehensive Listings:** Detailed views for hospitals (with tabs for Doctors, Lab, Pharmacy, etc.) and doctors (profile, schedule).
- **Appointment Lifecycle:** From browsing doctors to booking and confirming slots.
- **Inventory & Billing:** Modules for pharmacy stock management, lab test tracking, and both OPD/IPD billing.
- **Responsive & Accessible:** Mobile-first design ensuring usability across all devices, with a focus on accessibility (semantic HTML, ARIA labels).
- **Modern UI/UX:** Clean interface with smooth animations, toast notifications, and intuitive modals.

---

## 🏗️ Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla ES6)
- **Icons & Fonts:** Font Awesome 6.4.0, Google Fonts
- **Styling Approach:** Modular CSS with Custom Properties (Variables) and a BEM-like naming convention.
- **Core Philosophy:** Mobile-first, responsive design with a focus on performance and maintainability.

## 🔍 Project Overview

**MEDFIND** is a comprehensive web application designed to simplify healthcare discovery and management. Inspired by platforms like `bddoctors.com`, it serves as a centralized healthcare ecosystem where users can:

- 🔎 Find hospitals, doctors, and diagnostic labs based on location
- 👤 View detailed profiles with specialties, ratings, and availability
- 📅 Book appointments seamlessly with real-time slot selection
- 📊 Access medical history, reports, and prescriptions
- 🏥 Manage hospital operations (Admin, Pharmacy, Lab, Billing)

The platform features **six distinct portals** (Patient, Doctor, Admin, Pharmacy, Lab, Billing) ensuring tailored experiences for every user role. Built with vanilla HTML, CSS, and JavaScript, it demonstrates a modular, scalable, and production-ready frontend architecture.

- **Advanced Filtering:** Filter by specialty, rating, facilities, and distance
- **Role-Based Dashboards:** Dedicated interfaces for all user types
- **Appointment Management:** End-to-end booking flow with confirmation

- **Patient Portal:** Medical history, prescriptions, appointment tracking
- **Admin Panel:** User management, hospital verification, system reports
- **Pharmacy Module:** Medicine inventory with expiry alerts
- **Laboratory System:** Test tracking and report uploads
- **Billing Suite:** OPD/IPD billing with multiple payment options
- **Ward Management:** Bed availability (ICU/General/Cabin)

### 🎨 UI/UX Excellence
- **Mobile-First Design:** Flawless experience across all devices
- **Smooth Animations:** Hover effects, modal transitions, loading states
- **Toast Notifications:** Real-time feedback system
- **Accessibility:** Semantic HTML, ARIA labels, keyboard navigation

export function initLocation() {
    const citySelect = document.getElementById('city');
    const areaSelect = document.getElementById('area');
    
    citySelect.addEventListener('change', async (e) => {
        const areas = await fetchAreas(e.target.value);
        populateAreas(areas);
        localStorage.setItem('selectedCity', e.target.value);
    });
}

// notification.js - Toast System
export function showNotification(message, type = 'info') {
    const toast = document.createElement('div');
    toast.className = `toast toast-${type}`;
    toast.textContent = message;
    toast.style.animation = 'slideIn 0.3s ease';
    document.body.appendChild(toast);
    setTimeout(() => toast.remove(), 3000);
}
